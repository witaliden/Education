# Education

### Application provides access to information about employee's self improvement activity.

## Technologies

1. Java 17.
2. Spring Boot 3.1.6.
3. Spring Data (JPA/Hibernate).
4. Liquibase.

## Design approaches
- Microservices architecture
- REST
